mana - analysis of mRNA manufacture quality using Oxford Nanopore Sequencing

TOOL OPTIONS:
	Required:
	-b                          Input BAM file containing ONT reads aligned to plasmid sequence used to generate mRNA.
	-f                          Input reference FASTA file of the plasmid sequence used to generate mRNA.

	Optional:
	-h                          Display this help usage message
	-o                          Output directory. Default to "outputs".
	--ecoli                     Input FASTA file of the bacterium used for plasmid propagation. Default will be E.coli K12 ASM584v2.
	--plasmid                   Plasmid analysis
	--mrna                      mRNA analysis
	-p1                         Start coordinate of mRNA.
	-p2                         Last coordinate of mRNA before PolyA tract.


USAGE EXAMPLE:
	$ python3 mana.py -b sample.bam -f plasmid_gfp.fasta

OUTPUTS:
	report.txt            Output TEXT file report describing plasmid or mRNA quality.
